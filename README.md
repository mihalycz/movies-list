# movies list
Movies list test module. **!!! module is in the middle of development, it works, but markup is not ready yet.** 

```sh
npm i
```

**dev:**

```sh
npm run dev
```
To see demo page please open **dev/index.html**


**build:**

```sh
npm run build
```
To see demo page please open **build/index.html**

**to run unit tests:**

```sh
npm run test
```
